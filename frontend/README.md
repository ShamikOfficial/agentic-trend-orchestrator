# Frontend App

Production frontend foundation for `agentic-trend-orchestrator`.

## Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- shadcn/ui

## Run Locally
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## API Base URL
Set `NEXT_PUBLIC_API_BASE_URL` when needed. Default is:

`http://127.0.0.1:8000/api/v1`

## Current Goal
Build the product UI modules and connect them to backend APIs defined in `docs/api-contract.md`.
